## 🎯 三大核心目标

### 目标一：Windows 系统管理助理
**愿景**: 像真人助理一样管理 Windows 系统

**能力清单**:
- [ ] 文件/目录管理（整理、归档、搜索）
- [ ] 软件安装/卸载/更新
- [ ] 系统监控（CPU/内存/磁盘）
- [ ] 定时任务管理
- [ ] 网络诊断
- [ ] 开机启动项管理
- [ ] 环境变量管理
- [ ] PowerShell 脚本执行

**验收标准**:
> 用户说"帮我清理下载文件夹，把超过30天的文件移到归档目录"，助理能独立完成。

---

### 目标二：完整 Agent 项目组
**愿景**: 多 Agent 协作，像一个小团队

**角色设计**:
| Agent | 职责 | 触发条件 |
|-------|------|----------|
| **项目经理** | 拆解任务、分配工作、汇总结果 | 复杂任务 |
| **研究员** | 搜索信息、阅读文档、总结报告 | 需要调研 |
| **程序员** | 写代码、调试、代码审查 | 开发任务 |
| **系统管理员** | 执行命令、管理系统、监控状态 | 系统操作 |
| **写手** | 撰写文档、邮件、报告 | 内容创作 |

**协作流程**:
```
用户请求 → 项目经理拆解 → 分配给各 Agent → 并行执行 → 汇总结果 → 反馈用户
```

**验收标准**:
> 用户说"调研一下最新的地源热泵技术，写份报告"，Agent 团队自动分工完成。

---

### 目标三：自主进化
**愿景**: 能自我学习、自我改进、自我扩展

**进化维度**:
- [ ] **技能进化**: 发现新需求 → 创建 Skill → 沉淀到仓库
- [ ] **记忆进化**: 从对话中提取知识 → 更新 MEMORY.md
- [ ] **工具进化**: 发现常用操作 → 封装成脚本/工具
- [ ] **流程进化**: 重复任务 → 自动化 workflow
- [ ] **自我诊断**: 发现问题 → 自我修复或报告

**进化触发器**:
- 同一类任务被执行 3 次以上 → 封装成 Skill
- 用户纠正错误 → 更新记忆，避免再犯
- 发现更好工具 → 替换旧方案

**验收标准**:
> 用户使用一个月后，助理比刚开始时更懂用户，更高效。
