## ✅ Edge 配置方案

### 方案 A：自动检测（推荐）
OpenClaw 会自动检测系统默认浏览器，Edge 在检测顺序里排第 3（Chrome → Brave → **Edge** → Chromium → Canary）。

如果 Edge 已经是你的默认浏览器，直接用就行，无需额外配置。

### 方案 B：手动指定 Edge 路径
如果想明确指定用 Edge，在 `~/.openclaw/openclaw.json` 添加：

```json5
{
  browser: {
    enabled: true,
    defaultProfile: "openclaw",  // 或 "chrome"
    executablePath: "C:\\Program Files (x86)\\Microsoft\\Edge\\Application\\msedge.exe",
  },
}
```

## 🎯 两种使用方式

| 方式 | 说明 | 适合场景 |
|------|------|----------|
| **`openclaw` 模式** | 启动一个独立的 Edge 实例（橙色边框），与你的日常浏览隔离 | 自动化任务、测试、安全考虑 |
| **`chrome` 模式** | 通过扩展控制你现有的 Edge 标签页 | 想用日常浏览的 Edge，需要登录状态 |
